Como funciona o monitoramento de processos?

Um resumo visual do fluxo — só para alinhar o que vamos construir

1

Cadastro do processo

O advogado cadastra o número CNJ do processo no Juris AI (ex: 0001234-56.2024.8.26.0100). Só isso — o sistema cuida do resto.

↓
2

Robô verifica automaticamente

A cada hora (ou no horário configurado), um script Python consulta a API do Jusbrasil perguntando: "tem alguma movimentação nova neste processo?". O Jusbrasil já agrega dados de +100 tribunais (TJSP, TRT, STJ, TRFs, DJEN, etc.).

↓
✅ Tem novidade

Juris AI salva a movimentação no banco e dispara alertas instantâneos para o advogado.

— Sem novidade

Nada a fazer. O robô dorme e verifica novamente no próximo ciclo.

↓
3

Alerta chega ao advogado

O advogado recebe uma notificação com o resumo da movimentação — sem precisar entrar no portal do tribunal. Pode ser via painel, e-mail ou WhatsApp.

4

IA traduz o juridiquês (diferencial)

Além de alertar o advogado, a IA pode enviar ao cliente uma mensagem simples no WhatsApp: "Seu processo teve uma atualização hoje: o juiz marcou uma audiência para 20/05."

Entendeu o fluxo? Responda no terminal com dúvidas ou diga "entendi" para continuar.
